# Financial-Transaction-Fraud-Detection-Data-Lake
04/18/2026 - 05/22/2026
Developed a scalable data pipeline that ingests raw financial transaction data into a data lake, performs feature engineering for fraud detection, and generates predictions using machine learning models. 

Pipeline Architecture
1.	Ingestion Layer – Load raw transaction data
2.	Processing Layer – Clean and transform data
3.	Feature Engineering – Extract fraud-related signals
4.	Modeling Layer – Train and evaluate ML models
5.	Prediction Layer – Generate fraud predictions
